风险评估与缓冲策略

Splat 渲染适配风险

当前 SplatRenderAdapter 是 stub，若 3DGS 渲染无法在 D6-D7 跑通，降级方案：使用 GLTF 软件渲染 (Tesla 示例已通) 做 Demo，保留 Splat 接口。

NL 解析精度不足

复杂镜头描述的意图识别可能不够准确。缓冲：D4-D5 留有测试集打磨时间；最坏情况用预设模板 + 下拉选择替代纯 NL。

Agent Flow 集成复杂度

LangGraph 或自定义 Agent 编排可能遇到状态管理和错误恢复问题。缓冲：JY 在 D4 前完成框架；如果 LangGraph 不稳定，退回到简单的 Chain 模式。

联调窗口紧张

M2 (4/5) 是关键里程碑，三人模块第一次合流。建议：4/2 对齐接口 Schema 后，每天 15 分钟 stand-up 同步阻塞点。
